# 🔻 API Support

Gigapipe is powered by an extended version of [qryn](https://qryn.dev) and supports the following APIs

<Tabs>
            <Tab label="Loki">
            | **Name**             | **Type**         | **QRYN**      | **QRYN Cloud**      | **APIs.**                  |   
            |----------------------|------------------|---------------|---------------------|----------------------------|
            | Push                 | POST             | 🟢             | 🟢             | `/loki/api/v1/push`        |
            | Query                | GET              | 🟢             | 🟢             | `/loki/api/v1/query`       |
            | Query Range          | GET              | 🟢             | 🟢             | `/loki/api/v1/query_range` | 
            | Labels               | GET              | 🟢             | 🟢             | `/loki/api/v1/label`       | 
            | Label Values         | GET              | 🟢             | 🟢             | `/loki/api/v1/label/name/values` | 
            | Tail                 | GET              | 🟢             | 🟢             | `/loki/api/v1/tail`        | 
            | Ready                | GET              | 🟢             | 🟢             | `/ready`                   | 
            
            
            ##### Compatible Agents
            
            ⭐ Any Loki compatible client can be used with qryn without modifications
            
            📦 [Grafana Alloy](https://grafana.com/docs/alloy/latest/)<br/>
            📦 [Grafana Agent](https://grafana.com/docs/agent/latest/)<br/>
            📦 [Vector](https://vector.dev/)<br/>
            📦 [Opentelemetry](https://github.com/metrico/otel-collector)<br/>
            📦 [Telegraf](https://www.influxdata.com/time-series-platform/telegraf/)<br/>
            📦 [Fluentd](https://www.fluentd.org/)<br/>
            📦 [Logstash](https://opensearch.org/docs/latest/tools/logstash/index/)<br/>
            </Tab>
